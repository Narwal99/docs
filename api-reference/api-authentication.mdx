<Note>
The HOST address for the API server is:
```
https://api.gpuhub.com/
```
</Note>

## Authentication

Create a token at Console → Settings → Tokens and use it for authentication.

![](/images/setting-token.png)

```javascript
headers = {"Authorization": "your token"}
```
