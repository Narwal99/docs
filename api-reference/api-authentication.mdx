<Note>
The HOST address for the API server is:
```
https://api.gpuhub.com/
```
</Note>

## Authentication

Create a token at [Console](https://api.gpuhub.com/console/center/settings/token) and use it for authentication.

```javascript
headers = {"Authorization": "your token"}
```
